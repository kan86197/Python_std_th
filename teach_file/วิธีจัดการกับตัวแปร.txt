สวัสดีครับ %n% วันนี้เราจะมาเรียนเรื่อง วิธีจัดการกับตัวแปรกันครับ %w%
การย้ายและการคัดลอกข้อมูลระหว่างตัวแปร
นอกจากการที่เราจะทำการให้ค่ากับตัวแปรโดยตรงผ่านเครื่องหมาย = แล้ว
เรายังสามารถให้ค่าตัวแปรได้โดยการรับค่าจากตัวแปรอื่นอีกด้วยครับ ยกตัวอย่างเช่น
first_var = 10
second_var = first_var  เป็นต้น %w%
ในที่นี้จะเห็นได้ว่าบรรทัดแรกนั้นเป็นการมอบค่าตัวเลข 10 ให้กับตัวแปร first_var
ส่วนในบรรทัดต่อมาเป็นการบอกไพธอนว่าให้ตั้งตัวแปร second_var มีค่าเท่ากับ first_var %w%
ซึ่งการกำหนดข้อมูลให้กับตัวแปรนั้นมีข้อควรระวังคือการเขียนข้อมูลไม่สามารถสลับฝั่งได้เนื่องจากจะทำให้ความหมายเปลี่ยนไป
เช่น first_var = second_var จะไม่เหมือนกับ second_var = first_var
เนื่องจากตัวแรกที่เขียนว่า first_var = second_var นั้นหมายความว่าให้นำค่า second_var ไปใส่ในตัวแปร first_var
ส่วนตัวที่สอง second_var = first_var คือการนำค่า first_var ใส่ในตัวแปร second_var %w%
นอกจากนี้กรณีการกำหนดข้อมูลให้กับตัวแปรก็ไม่สามารถเขียนให้ตัวแปรอยู่ทางด้านขวาได้
ยกตัวอย่างเช่น ลองพิมพ์ 5 = b %p%
%ca%
จะเห็นได้ว่าตัวไพธอนเองโชว์ค่า error เนื่องจากเราไม่สามารถกำหนดค่าในตัวแปร b ให้กับตัวเลขได้ %w%
สรุปคือฝั่งซ้ายของเครื่องหมาย = จะต้องเป็นตัวแปรเท่านั้น %w%
นอกจากนี้ไพธอนเองยังสามารถกำหนดค่าของตัวแปรได้ทีละมากกว่าหนึ่งตัวภายในครั้งเดียว
เช่น a,b,c = 10,20,30 เป็นต้น %w%
แต่ทว่าการกำหนดตัวแปรแบบหลายตัวพร้อมกันเช่นนี้ จำนวนตัวแปรจำเป็นทีจะต้องเท่ากับจำนวนตัวเลขที่ถูกป้อนด้วยไม่เช่นนั้นไพธอนจะส่งค่า error
ยกตัวอย่างเช่น a,b,c = 10,20,30,40 เช่นนี้ ตัวไพธอนจะฟ้อง error กลับมาในหน้าจอของเรา %w%
นอกจากนี้หลังจากที่ตัวแปรถูกกำหนดค่าไปแล้ว ตัวแปรก็จะสามารถที่จะใส่ค่าเข้าไปใหม่ได้ เช่น
name = 'empty'
name = '%n%'
เมื่อเราสั่ง print(name) ค่าที่เราจะได้ก็คือ %n%
เนื่องจากไพธอนจะทำการรันโค้ดเป็นบรรทัดเรียงกันไป ทำให้ค่าล่าสุดของ name คือ %n%
ส่วนตัว 'empty' นั้นก็จะถูก '%n%' เขียนทับไปนั่นเอง %w%
นอกจากนี้ในภาษาไพธอนจะสามารถกำหนดค่าตัวแปรโดยใช้ค่าเดิมมาเป็นส่วนประกอบในการคำนวณด้วยก็ได้ เช่น
a = 2
a = a+2 หรือ a += 2
ซึ่งจะทำให้ a มีค่าเป็น 4 นั่นเอง %w%
่ต่อมาเราจะมาพูดถึงการสลับข้อมูลในตัวแปรกันครับ
โดยปกติแล้วภาษาคอมพิวเตอร์ทั่วไปถ้าเราต้องการจะสลับข้อมูลของตัวแปร
เราจำเป็นที่จะต้องสร้างตัวแปรเพิ่มขึ้นมาอีกหนึ่งตัวเพื่อเก็บข้อมูลพักเอาไว้แล้วค่อยย้ายข้อมูลอีกรอบหนึ่ง เช่น
a = 10
b = 15
c = a
a = b
b = c
เช่นนี้ a ก็จะมีค่าเป็น b และ b มีค่าเท่ากับ a เป็นต้น %w%
แต่ในภาษาไพธอนเองมีความสามารถที่จะสลับค่าตัวแปลได้เลย เช่น
a,b = b,a
a,b,c = c,a,b
โดยข้อมูลจะถูกย้ายจากตัวแปรทางขวาไปยังตัวแปรทางซ้ายเทียบตามตำแหน่งครับ %w%
ถ้าเราอยากจะลบตัวแปรก็จะสามารถทำได้โดยใช้คำสั่ง del(ชื่อตัวแปร) เช่น
del(a) , del(varia) เป็นต้น %w%
ซึ่งวันนี้ก็จะขอจบการสอนไว้เพียงเท่านี้ครับ
